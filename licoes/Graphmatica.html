Microsoft Mathematics

    O Microsoft Mathematics era um programa educativo da Microsoft, descontinuado em 2025, que servia como uma ferramenta para resolver problemas de matemática e ciências. Ele incluía uma calculadora gráfica completa e outras ferramentas para ajudar estudantes e professores a entenderem conceitos matemáticos. A Microsoft substituiu o programa pelo Microsoft Math Solver, disponível para web e dispositivos móveis, com funcionalidades semelhantes.

O que era e como funcionava o Microsoft Mathematics

O software original para Windows, Microsoft Mathematics, funcionava como uma poderosa calculadora com uma interface intuitiva. 

Ele permitia: Calculadora gráfica: Plotava gráficos 2D (em coordenadas cartesianas e polares) e 3D (em coordenadas cartesianas, cilíndricas e esféricas). 

    Resolução passo a passo: Oferecia soluções detalhadas para equações, desigualdades, integrais, derivadas e outros problemas, orientando o usuário no processo. 

    Conjunto de ferramentas: Disponibilizava outras ferramentas úteis, como: Solucionador de triângulos: Para explorar as relações entre ângulos e lados. 

    Conversor de unidades: Para converter diferentes medidas (comprimento, área, volume, etc.). Biblioteca de fórmulas: Com mais de 100 equações e fórmulas comuns.

Suplemento para Word e OneNote

Além do programa original e da versão atual, a Microsoft também ofereceu um suplemento para o Word e OneNote. 

Este complemento permitia: Criar gráficos e resolver equações: As funcionalidades da calculadora eram integradas diretamente aos documentos. Simplificar expressões: Ajudava a trabalhar com expressões algébricas dentro do ambiente do Word e OneNote.

Como funciona

O Microsoft Mathematics permite que resolvam equações, com indicação passo a passo do que está a ser feito, desenho de gráficos 2D e 3D, conversão de unidades e resolução de problemas com triângulos. 

     Na sua interface principal devem apenas colocar as equações que pretendem ver resolvidas e carregarem no botão Enter. Podem também colocar ai simples operações aritméticas que o Microsoft Mathematics tratará de as resolver e apresentar-vos o resultado.

    A apresentação dos gráficos permite que tenha uma visão diferente das equações que necessitam resolver e ajuda-vos a compreender melhor o que estão a calcular. No caso dos gráficos 3D podem mover e rodar esses gráficos para que tenham uma real perspectiva do gráfico. 

     Uma outra forma muito interessante que colocarem os dados no Microsoft Mathematics é através da escrita manual. Se tiverem um equipamento que suporte a escrita directa no ecrã, podem introduzir os dados directamente disse forma.
